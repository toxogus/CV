## 🚀 Projects

### GPSFramework — GlobalCLIP-PatchCore-SAM for Zero-Shot Anomaly Detection
*Smart Factory Capstone Design*

**Overview:** This three-stage, zero-shot framework facilitates immediate deployment on novel industrial products by integrating vision-language models with classical anomaly detection and prompt-based segmentation without requiring category-specific training.

- **Stage 1 (GlobalCLIP):** By utilizing an ensemble of six autoencoders alongside a dual-backbone CLIP architecture, the system generates coarse anomaly scores through Adaptive Region Selection and Adaptive Anomaly Weighting.
- **Stage 2 (PatchCore):** This stage employs a WideResNet-50 feature extractor combined with a coreset-reduced memory bank to achieve efficient, patch-level anomaly scoring via nearest-neighbor analysis.
- **Stage 3 (SAM):** The Segment Anything Model acts as a final refinement layer, utilizing prompt-guided inputs to produce precise, pixel-level localization of detected defects.
- **Benchmark:** Evaluated on the MVTec AD dataset, the model demonstrates robust performance, achieving competitive image-level AUROC and pixel-level PRO scores in a fully zero-shot setting.

## 📚 Seminars & Paper Reviews

### OneFormer: One Transformer to Rule Universal Image Segmentation
This paper presents OneFormer, a universal image segmentation framework that utilizes a task-conditioned joint training strategy to unify semantic, instance, and panoptic segmentation tasks within a single model trained on a single dataset, achieving state-of-the-art performance while significantly reducing resource requirements.

### LECLIP: Boosting Zero-Shot Anomaly Detection With Local Enhanced CLIP
LECLIP (Local Enhanced CLIP) improves zero-shot anomaly detection by addressing CLIP's limitation of focusing on global embeddings rather than local anomaly regions, introducing a local alignment (LA) module that aligns image blocks with learnable text embeddings, plus a training-free echo-attention (EA) mechanism that captures both global and local visual details. It achieves superior performance across 15 industrial and medical datasets.
