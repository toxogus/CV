## 🚀 Projects

### GPSFramework — GlobalCLIP-PatchCore-SAM for Zero-Shot Anomaly Detection
*Smart Factory Capstone Design*

A three-stage, zero-shot framework for immediate deployment on novel industrial products, integrating vision-language models with classical anomaly detection and prompt-based segmentation — no category-specific training required.

- **Stage 1 (GlobalCLIP):** Ensemble of six autoencoders with a dual-backbone CLIP architecture, generating coarse anomaly scores via Adaptive Region Selection and Adaptive Anomaly Weighting.
- **Stage 2 (PatchCore):** WideResNet-50 feature extractor with a coreset-reduced memory bank for efficient patch-level anomaly scoring via nearest-neighbor analysis.
- **Stage 3 (SAM):** Segment Anything Model as a refinement layer for precise, pixel-level defect localization using prompt-guided inputs.
- **Benchmark:** Evaluated on MVTec AD, achieving competitive image-level AUROC and pixel-level PRO scores in a fully zero-shot setting.
- **Outcomes:**
  - 📄 Patent (under review): *Zero-shot Anomaly Detection and Localization in Industrial Environments using GlobalCLIP Patch Tokens* — Ref. R-2026-0537-KR-1
  - 💻 SW Registration: *GPTCLIP — Industrial Zero-shot Anomaly Detection and Localization using GlobalCLIP Patch Tokens* — Corp. Reg. No. 110171-0029501

## 📚 Seminars & Paper Reviews

**OneFormer: One Transformer to Rule Universal Image Segmentation**
A universal image segmentation framework using task-conditioned joint training to unify semantic, instance, and panoptic segmentation in a single model trained on a single dataset — achieving state-of-the-art performance with reduced resource requirements.

**LECLIP: Boosting Zero-Shot Anomaly Detection With Local Enhanced CLIP**
Improves zero-shot anomaly detection by addressing CLIP's bias toward global embeddings, introducing a local alignment (LA) module that aligns image blocks with learnable text embeddings, plus a training-free echo-attention (EA) mechanism capturing both global and local visual details — achieving superior performance across 15 industrial and medical datasets.
